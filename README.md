# 🚀 Workly – Internship Platform Frontend
Workly is a modern internship platform designed to connect students with meaningful opportunities while helping companies discover skilled talent efficiently.
This repository contains the **frontend application** built using React.js.

## 🌟 Features

### 👨‍🎓 Student Side
* Browse internships
* Apply to opportunities
* Track application status
* Manage profile and resume

### 🏢 Company Side
* Post internship opportunities
* Manage listings
* View and shortlist candidates

### 🛠 Admin Panel
* Manage users and companies
* Monitor platform activity
* View analytics dashboard

## 🧑‍💻 Tech Stack
* Frontend: React.js (Vite)
* Styling: Tailwind CSS
* Routing: React Router
* State Management: Context API
* HTTP Requests: Axios (mock setup)

## 📂 Project Structure

src/
├── components/      # Reusable UI components
├── pages/           # Application pages (Student, Company, Admin)
├── context/         # Global state management
├── routes/          # Routing configuration
├── assets/          # Images & static files
├── App.jsx          # Main app component
└── main.jsx         # Entry point

## ⚙️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/Kavita635/Workly-frontend.git

### 2. Navigate to project folder
cd Workly-frontend

### 3. Install dependencies
npm install

### 4. Run the development server
npm run dev

### 5. Open in browser
http://localhost:5173/
